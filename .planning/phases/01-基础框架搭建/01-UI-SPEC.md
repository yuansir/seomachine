## Interaction Patterns

### Onboarding Flow
1. First launch → Detect first-use (no API keys in settings)
2. Show welcome dialog → "欢迎使用 SEO Machine"
3. Step 1: Enter Claude API Key → validate non-empty → next
4. Step 2: Enter DataForSEO Key (optional) → skip option → next
5. Step 3: WordPress Config (optional) → skip option → next
6. Complete → show main app shell

### Settings Page Interactions
- **Save:** Auto-save with 500ms debounce OR manual save button
- **Test Connection:** Show loading → success/error toast
- **Password visibility:** Toggle eye icon
- **Validation:** Real-time, show error below field

### Navigation Interactions
- **Sidebar item click:** Instant highlight, content load
- **Active indicator:** Left border accent color (blue-500)
- **Hover:** Background lighten

### Error Handling
- **API error:** Toast with specific message + retry option
- **Network error:** "网络连接失败，请检查网络" + retry button
- **Validation error:** Inline error below input, red border
